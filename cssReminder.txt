color: Sets the color of text.
font-size: Specifies the size of the font.
font-family: Defines the font family for text.
font-weight: Specifies the thickness of characters in a text.
font-style: Sets the style of the font (italic, oblique, normal).
text-decoration: Adds decoration to text (underline, overline, line-through).
text-align: Aligns text horizontally.
line-height: Sets the height of a line of text.
letter-spacing: Adjusts spacing between characters in text.
word-spacing: Adjusts spacing between words in text.
text-transform: Controls the capitalization of text.
background-color: Sets the background color of an element.
background-image: Sets an image as the background of an element.
background-position: Specifies the starting position of a background image.
background-repeat: Defines how a background image is repeated.
background-size: Sets the size of a background image.
border: Specifies the border properties of an element.
border-color: Sets the color of the border.
border-style: Sets the style of the border (solid, dashed, dotted, etc.).
border-width: Sets the width of the border.
border-radius: Rounds the corners of an element's border.
padding: Sets the padding area around the content of an element.
margin: Sets the margin area outside the border of an element.
width: Specifies the width of an element.
height: Specifies the height of an element.
display: Defines how an element is displayed (block, inline, inline-block, none, etc.).
position: Specifies the positioning method of an element (static, relative, absolute, fixed).
top, right, bottom, left: Positions an element relative to its containing element.
float: Aligns an element to the left or right within its container.
clear: Specifies whether an element can be next to floating elements.
overflow: Specifies what happens if content overflows its container.
z-index: Specifies the stack order of an element.
opacity: Sets the transparency level of an element.
box-shadow: Adds shadow effects to an element.
text-shadow: Adds shadow effects to text.
transform: Applies 2D or 3D transformations to an element.
transition: Specifies the transition effects on an element's properties.
animation: Defines animations on an element.
cursor: Specifies the type of cursor to be displayed when hovering over an element.
pointer-events: Specifies whether an element should respond to pointer events.
user-select: Controls the user's ability to select text.
white-space: Specifies how white space inside an element is handled.
list-style: Sets the style of list markers.
outline: Sets the outline properties of an element.
text-overflow: Specifies how overflowed content in a text should be signaled to the user.
box-sizing: Defines how the total width and height of an element are calculated.
content: Inserts content before or after an element's content.
flex: Specifies flexible lengths and the ability of an element to alter its dimensions.
align-items: Aligns flex items along the cross axis of the flex container.
justify-content: Aligns flex items along the main axis of the flex container.
